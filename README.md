# BlogCraft

BlogCraft is an AI-assisted blog writing app with a React frontend and a FastAPI backend.

## Project Structure

- `fe/` - Frontend (React + TypeScript + Vite)
- `be/` - Backend (FastAPI + LangGraph + LLM integrations)

## Quick Start

1. Start backend from `be/`
2. Start frontend from `fe/`
3. Open the frontend URL shown by Vite

## Deployment

- Frontend: Vercel
- Backend: Render

Set frontend env variable:

- `VITE_API_BASE_URL=https://blog-craft.onrender.com/api`
